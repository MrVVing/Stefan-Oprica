# Sales Analysis Project

## Description
This mini SQL project demonstrates skills in table creation, data insertion, and basic analysis using SQL queries.

## Project Structure
- `sales_data.csv` - example dataset
- `sales_analysis.sql` - SQL script with tables and queries
- `README.md` - project explanation

## What I did
1. Created the `Sales` table.
2. Inserted example data.
3. Wrote queries for:
   - Total sales per product
   - Total sales per region
   - Top 3 best-selling products

## Skills Demonstrated
- Creating and populating SQL tables
- Querying with `GROUP BY`, `SUM()`, `ORDER BY`
- Simple data analysis

## How to run
1. Open SQL Server Management Studio or any SQL client.
2. Run `sales_analysis.sql` to create the table and insert the data.
3. Execute the queries to view analysis results.
